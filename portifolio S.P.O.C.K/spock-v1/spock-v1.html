Como Fizemos:
Criamos um protótipo de mão robótica usando materiais simples e acessíveis. A estrutura foi feita em madeira, com dedos de cabos de vassoura conectados por dobradiças para simular articulações. Elásticos foram usados como tendões, trazendo os dedos de volta à posição inicial após o movimento. Para controlar os movimentos, utilizamos uma luva equipada com potenciômetros deslizantes, conectados estrategicamente para capturar os movimentos dos dedos. Esses sensores enviam sinais para um Arduino Uno, que aciona os servomotores da mão robótica, reproduzindo os movimentos da luva. Esse projeto mostra como materiais comuns podem ser usados para desenvolver robótica funcional.
