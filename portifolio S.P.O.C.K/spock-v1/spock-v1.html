Como Fizemos:
Utilizamos os materiais listados nas imagens para realizar o projeto.
Para criar o protótipo de uma mão robótica usando materiais acessíveis e simples, foram utilizados madeira, dobradiças, elásticos e potenciômetros deslizantes. A estrutura da mão foi montada em madeira, que fornece uma base resistente e leve. Para os dedos, utilizamos cabo de vassoura conectados por pequenas dobradiças que simulam as articulações dos dedos, permitindo que os cabos se movam como uma mão real.
Elásticos foram fixados na parte traseira dos dedos para simular os tendões, de forma que, ao serem tensionados, eles trazem os dedos de volta à posição original. Esses elásticos proporcionam a flexibilidade e a resistência necessária para o retorno dos dedos, permitindo que o movimento seja repetido. Para controlar os movimentos, foi projetada uma luva equipada com potenciômetros deslizantes, que funcionam como sensores de movimento.
Cada potenciômetro foi colocado na luva em posições estratégicas correspondentes a cada dedo, de modo que ao dobrar um dedo da mão, o potenciômetro deslizante é acionado, registrando a intensidade do movimento.
Esses potenciômetros estão conectados a um microcontrolador, como o Arduino Uno, que lê os valores de cada potenciômetro e os converte em sinais para acionar pequenos servomotores ou outros mecanismos na mão robótica. Dessa forma, quando um dedo na luva é dobrado, o dedo correspondente na mão robótica se move de maneira similar. Esse sistema simples e acessível permite desenvolver o conceito básico de uma mão robótica funcional, mostrando como materiais comuns podem ser usados para prototipagem na robótica.
