Primeiro protótipo

Spock V1 é o primeiro protótipo desenvolvido pelo grupo, construído em madeira, ideal para quem deseja aprender sobre robótica de maneira acessível e econômica.

Primeira versão do Spock

Aprenda a criar seu próprio "Spock" usando materiais simples e acessíveis, como cabo de vassoura, madeira, potenciômetros deslizantes, Arduino e servomotores.
Primeiro teste dos servomotores
Código para o controle dos servos

Por Quê?

Baixo custo : Usando materiais fáceis como madeira e cabo de vassoura, o projeto permite que você aprenda sobre robótica sem precisar investir em componentes caros, tornando-o uma opção econômica para iniciantes.

Simplicidade de montagem;

Aprendizado prático de robótica.

Esse protótipo fornece uma introdução a conceitos essenciais de robótica, como controle de movimento e eletrônica básica, com componentes facilmente ilustrativos.

Principais Caracteristicas

Como o projeto utiliza materiais simples, é possível personalizar e modificar o protótipo com facilidade, possibilitando que você adapte o design para aprimorá-lo conforme seu conhecimento avançado.

Componentes como potenciômetros deslizantes, Arduino e servomotores, o projeto "Spock" é de fácil construção.
